# 📦 JavaScript Array Methods Project

An interactive JavaScript project built using **HTML**, **CSS**, and **JavaScript** to learn and practice the most commonly used **Array Methods**. This project allows users to dynamically add, remove, search, modify, and manipulate array elements through a simple and user-friendly interface.

## ✨ Features

* ➕ Add new elements using **Push**
* ➖ Remove last element using **Pop**
* ⬅ Remove first element using **Shift**
* ➡ Add element at the beginning using **Unshift**
* 🔤 Sort array elements
* 🔄 Reverse array
* ✂ Slice array
* 🪄 Splice array
* 🔍 Find element index using **IndexOf**
* ✅ Check element using **Includes**
* 🔗 Join array elements
* 📏 Display array length
* 🗑 Clear the complete array
* 📱 Responsive and modern UI
* ⚡ Dynamic array updates without page reload

## 📚 Array Methods Covered

| Method     | Description                             |
| ---------- | --------------------------------------- |
| push()     | Adds an element to the end of the array |
| pop()      | Removes the last element                |
| shift()    | Removes the first element               |
| unshift()  | Adds an element at the beginning        |
| sort()     | Sorts array elements                    |
| reverse()  | Reverses the array                      |
| slice()    | Returns a selected portion of the array |
| splice()   | Adds or removes array elements          |
| indexOf()  | Returns the index of an element         |
| includes() | Checks whether an element exists        |
| join()     | Converts array into a string            |
| length     | Returns total number of elements        |
